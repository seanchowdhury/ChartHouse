# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Courses

- `GET /api/courses`
- `POST /api/courses`
- `GET /api/courses/:id`
- `PATCH /api/courses/:id`
- `DELETE /api/courses/:id`

### Charts

- `GET /api/charts`
- `POST /api/charts`
- `GET /api/charts/:id`
- `PATCH /api/charts/:id`
- `DELETE /api/charts/:id`
